# MedsVerifier
A Consumer-Centric Blockchain Architecture for Low-Cost Pharmaceutical Authenticity Verification in Developing Regions

MSc Business Computing Dissertation — Staffordshire University, 2026

# Overview

MedsVerifier is a permissioned blockchain lightweight system that utilizes ECDSA P-256 cryptography and QR code-based drug authentication, targeted for consumers medicines authenticity verification on low-end Android devices in developing regions. The system is built with Hyperledger Fabric 2.5.0, with a bulk key synchronisation mechanism for offline verification and governance at the protocol level enforced at the chaincode layer by the regulator (NAFDAC).
